# End-to-End-Drug-Discovery-Pipeline

This project presents an end-to-end computational drug discovery pipeline targeting the EGFR protein in non-small cell lung cancer (NSCLC). It integrates key steps
such as target identification, structure preparation, molecular docking using AutoDock Vina, and Python-based parsing to rank compounds based on binding affinity. The
workflow also includes ADMET filtering using drug-likeness rules to identify safe and effective lead compounds, with results validated using known inhibitors like
Osimertinib and Afatinib.

The pipeline is designed to be modular, scalable, and aligned with real-world pharmaceutical workflows. It automates data processing, generates structured outputs
(CSV, plots, summaries), and supports protein–ligand interaction analysis using PyMOL for visualization. While the system demonstrates efficiency and accuracy in early
stage drug screening, it is limited to computational predictions and requires experimental validation for real-world application.
